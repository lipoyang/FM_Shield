# FM Shield

## 概要
GR-SAKURAでFM音源YM2203を制御するシールド基板とソースコード(ライブラリおよびサンプルアプリ)です。ライブラリはMMLの文字列データを再生します。MMLの仕様はN88-BASICのMMLのサブセットです。
